# Contributing Guide

Terima kasih atas minat Anda untuk berkontribusi pada modul praktikum ini!

## Cara Berkontribusi

### 1. Fork Repository

Fork repository ini ke akun GitHub Anda.

### 2. Clone Repository

```bash
git clone https://github.com/YOUR-USERNAME/modul-praktikum-react-native.git
cd modul-praktikum-react-native
```

### 3. Buat Branch Baru

```bash
git checkout -b fitur/nama-fitur
# atau
git checkout -b fix/nama-bug
```

### 4. Lakukan Perubahan

- Pastikan kode mengikuti style guide yang ada
- Tambahkan komentar jika diperlukan
- Test perubahan Anda

### 5. Commit Perubahan

```bash
git add .
git commit -m "feat: deskripsi singkat perubahan"
```

Format commit message:
- `feat:` untuk fitur baru
- `fix:` untuk perbaikan bug
- `docs:` untuk perubahan dokumentasi
- `style:` untuk formatting, semicolons, dll
- `refactor:` untuk refactoring kode
- `test:` untuk menambah/memperbaiki test

### 6. Push dan Buat Pull Request

```bash
git push origin fitur/nama-fitur
```

Lalu buat Pull Request di GitHub.

## Panduan Penulisan Tutorial

1. Gunakan Bahasa Indonesia yang baik dan benar
2. Sertakan contoh kode yang lengkap dan bisa dijalankan
3. Tambahkan penjelasan untuk setiap konsep baru
4. Sertakan screenshot jika membantu pemahaman
5. Akhiri dengan tugas praktikum
