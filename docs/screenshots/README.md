## Tips Mengambil Screenshot

### Android
- **Emulator**: Klik tombol kamera di toolbar
- **Device**: Power + Volume Down

### iOS
- **Simulator**: Cmd + S
- **Device**: Power + Volume Up

### Web
- **Chrome**: Ctrl/Cmd + Shift + P â†’ "Capture screenshot"
- **Firefox**: Ctrl/Cmd + Shift + S

## Catatan

- Pastikan screenshot jelas dan mudah dibaca
- Gunakan format PNG untuk kualitas terbaik
- Ukuran file maksimal 2MB per screenshot
- Jangan edit/manipulasi screenshot
